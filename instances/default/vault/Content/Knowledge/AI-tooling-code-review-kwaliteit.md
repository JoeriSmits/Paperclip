# AI-tooling verbetert code review kwaliteit (niet verlaagt het)

## Insight

Na 3 maanden Cursor gebruik bij Webbio: de code review gesprekken gaan nu over architectuur in plaats van syntax. AI vangt de makkelijke fouten op, waardoor de review aandacht vrijkomt voor de moeilijke vragen. Junior developers leren er meer van dan seniors — ze gebruiken het als interactieve documentatie.

Concrete observatie: onboarding van de laatste twee developers was een stuk sneller dan normaal. ROI zit niet in story velocity maar in aandachtsverdeling en onboarding-snelheid.

Sterkste quote: "De aandacht gaat nu naar het moeilijke, niet het eenvoudige."

## Joeri's Angle

Joeri observeert dit vanuit zijn rol als Head of Development / CTO bij Webbio. Hij had zelf ook de verwachting dat juniors blind code zouden accepteren — maar de code review bleek het vangnet. Dat is de echte productiviteitswinst: niet snellere code schrijven, maar betere discussies voeren.
